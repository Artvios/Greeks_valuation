# Greeks_valuation


This repository demonstrates the calculation of option Greeks and related hedging strategies using Python.
It includes:

- `Option_Greeks.ipynb` – a Jupyter notebook detailing the mathematical background and code for computing option Greeks.
- `greeks_summary.py` – a script that prints concise examples and insights for practical hedging and risk management.

## Usage

Run the summary script with Python 3:

```bash
python greeks_summary.py
```

The script prints several analyses covering delta hedging, theta decay, volatility impacts, and other risk considerations.

============================================================
RISK MANAGEMENT INSIGHTS
============================================================

1. DELTA RISK (Directional Risk)
-----------------------------------
• Delta measures directional exposure to underlying price
• Portfolio delta = sum of individual position deltas
• Delta-neutral portfolio: portfolio delta ≈ 0
• Rebalance frequency depends on gamma (delta sensitivity)

2. GAMMA RISK (Convexity Risk)
-----------------------------------
• Gamma measures how fast delta changes
• High gamma = frequent rebalancing needed
• Gamma risk highest for ATM options near expiration
• Long options: positive gamma (beneficial)
• Short options: negative gamma (risk)

3. THETA RISK (Time Decay Risk)
-----------------------------------
• Theta measures daily time decay
• Long options: negative theta (lose value daily)
• Short options: positive theta (gain value daily)
• Theta accelerates as expiration approaches
• ATM options have highest theta risk

4. VEGA RISK (Volatility Risk)
-----------------------------------
• Vega measures sensitivity to volatility changes
• Long options: positive vega (benefit from vol increase)
• Short options: negative vega (hurt by vol increase)
• Vega highest for ATM options with more time
• Implied volatility can be as important as direction

5. RHO RISK (Interest Rate Risk)
-----------------------------------
• Rho measures sensitivity to interest rate changes
• Generally smallest Greek for short-term options
• More important for long-term options (LEAPS)
• Call rho positive, put rho negative
